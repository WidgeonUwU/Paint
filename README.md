# Paint

Android-приложение для рисования, разработанное в рамках учебной практики в университете.  

## Описание проекта

Приложение предоставляет пользователю возможность создавать и редактировать графические рисунки на мобильном устройстве.  

Проект реализован как законченный продукт: от проектирования архитектуры до тестирования ключевых компонентов.

## Основная функциональность

- Рисование на холсте с использованием базовых инструментов
- Поддержка истории действий (Undo / Redo)
- Создание, сохранение и редактирование рисунков
- Экспорт изображений в графические форматы (PNG / JPEG)
- Хранение данных локально на устройстве

## Архитектура

Приложение построено с использованием принципов чистой архитектуры:

- MVVM (Model–View–ViewModel)
- Разделение UI, бизнес-логики и слоя данных
- Минимальная связность компонентов
- Чёткие интерфейсы взаимодействия между слоями

Такой подход упростил тестирование логики и позволил изолировать бизнес-логику от пользовательского интерфейса.

## Используемые технологии

- **Язык:** Kotlin
- **UI:** Jetpack Compose
- **Архитектура:** MVVM, Clean Architecture
- **Хранение данных:** Room (SQLite)
- **Сборка проекта:** Gradle
- **Тестирование:** JUnit

# Функциональный чек‑лист (регрессия) для Paint

## UI и навигация
- Экран галереи открывается, заголовок отображается корректно.
- Кнопка «+» создания рисунка доступна.
- Экран пустой галереи содержит кнопку создания.
- Переход в холст и возврат назад работают без ошибок.
- В портретной/альбомной ориентации элементы не перекрываются.

## Создание холста
- Диалог выбора шаблона открывается при создании холста.
- Пресеты создают холст с ожидаемыми размерами.
- Пользовательский размер принимает числовые значения.
- Нечисловые значения приводят к созданию холста по умолчанию.

## Инструменты рисования
- Выбор инструмента (карандаш/ластик/линейка) переключает режим.
- Карандаш рисует непрерывную линию.
- Ластик удаляет ранее нарисованное.
- Линейка рисует прямую между началом и концом жеста.

## Палитра и параметры
- Тап по цвету меняет текущий цвет.
- Двойной тап по цвету открывает выбор нового цвета и заменяет его в палитре.
- Диалог толщины изменяет толщину новых линий.
- Диалог прозрачности изменяет прозрачность новых линий.

## История и очистка
- Undo отменяет последнее действие.
- Redo восстанавливает отмененное действие.
- Множественные undo/redo работают корректно.
- Очистка удаляет весь рисунок.

## Сохранение и экспорт
- Сохранение открывает диалог имени и создает запись в галерее.
- Переименование рисунка работает из холста (double‑tap по названию).
- Экспорт PNG завершаетcя успешно и файл доступен в галерее устройства.
- Экспорт JPEG завершаетcя успешно и файл доступен в галерее устройства.
- При ошибке экспорта выводится сообщение без падения приложения.

## Галерея
- Поиск фильтрует список рисунков.
- Открытие рисунка из карточки загружает его на холст.
- Переименование в галерее сохраняется.
- Удаление рисунка удаляет его из списка.

## Граничные и нестандартные ситуации
- Смена ориентации не приводит к потере данных и крашу.
- Уход приложения в фон сохраняет текущее состояние.
- Длительная сессия рисования не приводит к зависаниям.

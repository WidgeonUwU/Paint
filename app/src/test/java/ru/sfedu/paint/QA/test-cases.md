# Функциональные тест‑кейсы для Paint

  

| ID тест‑кейса | Название                                            | Предусловия                                                                     | Шаги                                                                                                                            | Ожидаемый результат                                                                        | Приоритет |
| ------------- | --------------------------------------------------- | ------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------ | --------- |
| TC-001        | Создание нового холста из галереи (кнопка «+»)      | Приложение установлено и запущено, открыт экран галереи                         | 1. Нажать «+» в верхней панели                                                                                                  | Открывается экран холста, отображается диалог выбора шаблона                               | Высокий   |
| TC-002        | Создание нового холста из пустого состояния галереи | В галерее нет рисунков                                                          | 1. Нажать кнопку «Создать» на экране пустой галереи                                                                             | Открывается экран холста, отображается диалог выбора шаблона                               | Средний   |
| TC-003        | Создание холста по пресету из диалога шаблонов      | Открыт диалог выбора шаблона                                                    | 1. Выбрать любой пресет (карточка соотношения сторон)                                                                           | Холст создается с размерами пресета, диалог закрывается                                    | Высокий   |
| TC-004        | Создание холста с пользовательским размером         | Открыт диалог выбора шаблона                                                    | 1. Нажать «Пользовательский размер»<br>2. Ввести ширину и высоту (числа)<br>3. Нажать «ОК»                                      | Холст создается с указанными размерами                                                     | Высокий   |
| TC-005        | Обработка нечисловых размеров холста                | Открыт диалог пользовательского размера                                         | 1. Ввести нечисловые значения (например, «abc»)<br>2. Нажать «ОК»                                                               | Холст создается с размерами по умолчанию (1920×1080)                                       | Средний   |
| TC-006        | Рисование карандашом                                | Холст открыт                                                                    | 1. Выбрать инструмент «Карандаш»<br>2. Провести линию по холсту                                                                 | Линия появляется без разрывов и повторяет движение пальца                                  | Высокий   |
| TC-007        | Рисование линейкой                                  | Холст открыт                                                                    | 1. Выбрать инструмент «Линейка»<br>2. Провести жест рисования                                                                   | Рисуется ровная линия от точки начала до точки окончания                                   | Средний   |
| TC-008        | Рисование ластиком                                  | Холст содержит рисунок                                                          | 1. Выбрать инструмент «Ластик»<br>2. Провести по нарисованной линии                                                             | Линия удаляется по траектории ластика                                                      | Высокий   |
| TC-009        | Изменение толщины линии                             | Холст открыт                                                                    | 1. Нажать кнопку толщины (иконка «галочка»)<br>2. Изменить значение слайдера<br>3. Нажать «ОК»<br>4. Нарисовать линию           | Новая линия соответствует выбранной толщине                                                | Средний   |
| TC-010        | Изменение прозрачности (opacity)                    | Холст открыт                                                                    | 1. Нажать кнопку прозрачности (иконка «прозрачность»)<br>2. Изменить значение слайдера<br>3. Нажать «ОК»<br>4. Нарисовать линию | Новая линия имеет выбранную прозрачность                                                   | Средний   |
| TC-011        | Undo/Redo последнего действия                       | Холст содержит минимум одно действие рисования                                  | 1. Нажать «Undo»<br>2. Нажать «Redo»                                                                                            | Последнее действие отменяется и затем восстанавливается                                    | Высокий   |
| TC-012        | Очистка холста                                      | Холст содержит рисунок                                                          | 1. Нажать кнопку «Очистить» (иконка очистки)                                                                                    | Холст очищается полностью                                                                  | Средний   |
| TC-013        | Сохранение рисунка в галерею приложения             | Холст содержит рисунок                                                          | 1. Нажать кнопку сохранения (иконка «готово»)<br>2. Ввести имя<br>3. Нажать «Сохранить»                                         | Рисунок сохраняется и появляется в галерее приложения                                      | Высокий   |
| TC-014        | Экспорт изображения (PNG/JPEG)                      | Холст содержит рисунок                                                          | 1. Нажать кнопку экспорта (иконка «сохранить»)<br>2. Выбрать «PNG» или «JPEG»                                                   | Файл изображения создается в системной галерее/хранилище, отображается сообщение об успехе | Высокий   |
| TC-015        | Ошибка экспорта при сбое сохранения                 | Холст содержит рисунок, имитирован сбой сохранения (например, отсутствие места) | 1. Нажать кнопку экспорта<br>2. Выбрать формат                                                                                  | Экспорт не выполняется, отображается сообщение об ошибке                                   | Высокий   |

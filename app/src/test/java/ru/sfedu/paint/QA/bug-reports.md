# Отчеты о дефектах

## BUG-001: Undo возвращает текущее состояние вместо предыдущего
**Шаги воспроизведения:**
1. Открыть холст.
2. Нарисовать линию.
3. Нажать «Undo».

**Ожидаемый результат:** последняя линия удаляется, отображается предыдущее состояние.  
**Фактический результат:** состояние не меняется, возвращается текущее.  
**Окружение:** Android 14, Pixel 5 (эмулятор).  
**Приоритет:** Высокий

---

## BUG-002: Некорректная проверка сжатия JSON в автотесте
**Шаги воспроизведения:**
1. Запустить тест `JsonCompressionTest.compress_ValidJsonString_ReturnsCompressedByteArray`.

**Ожидаемый результат:** тест проходит для валидной строки.  
**Фактический результат:** тест падает, если размер результата не меньше исходного для маленьких данных.  
**Окружение:** Локальный запуск JUnit, Windows 11, Gradle.  
**Приоритет:** Низкий

---

## BUG-003: Двойной тап по названию не открывает переименование в новом рисунке
**Шаги воспроизведения:**
1. Создать новый холст.
2. Дважды нажать на название в верхней панели.

**Ожидаемый результат:** открывается диалог переименования.  
**Фактический результат:** диалог не открывается, имя остается «Новый рисунок».  
**Окружение:** Android 14, Pixel 5 (эмулятор).  
**Приоритет:** Низкий

---

## BUG-004: После очистки холста Undo не восстанавливает рисунок
**Шаги воспроизведения:**
1. Открыть холст и нарисовать несколько линий.
2. Нажать «Очистить».
3. Нажать «Undo».

**Ожидаемый результат:** восстанавливается состояние до очистки.  
**Фактический результат:** холст остается пустым, состояние не восстанавливается.   
**Окружение:** Android 12, Pixel 5 (эмулятор).  
**Приоритет:** Высокий
